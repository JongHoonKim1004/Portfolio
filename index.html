엠브레인 패널파워

Spring Boot + ReactJS

온라인 설문조사 엠브레인 패널파워 페이지입니다. ReactJS를 프론트엔드로 Spring Boot를 백엔드로 하여 개발한 사이트입니다.

메인 페이지에서 로그인과 로그아웃, 설문조사 참여, 마이페이지의 이동이 가능합니다. 페이지 이동을 원할하게 진행하기 위해 react-router-dom의 useParams, useNavigate, Route, Router, Link, Routes를 사용했습니다. 이메일 전송 시스템을 갖추고 있어, 비밀번호 재설정을 위한 인증번호 전송이 가능합니다.

Spring Security와 JSON Web Token을 적용했습니다. 토큰 저장과 설문조사 질문 저장을 위해 Redux를 사용했습니다.

개발 기간 : 2024/04/24 ~ 05/22

  • 설문조사
  • 사업자 페이지
  • 관리자 페이지

오지캠핑

Spring Legacy MVC

캠핑장 예약 사이트 오지캠핑입니다.

Spring Legacy와 JSP로 개발한 사이트입니다.

메인 페이지에서 로그인과 로그아웃, 회원가입, 캠핑장 검색, 마이페이지로 이동이 가능합니다. 공공데이터포털 API를 사용하여 캠핑장 목록을 호출하고 이를 Kakao Map API에 적용하였습니다. 실시간 예약 확인을 위해 캠핑장을 선택하고 예약 기간을 선택하면 해당 기간에 예약이 있는지
서버를 통해 확인합니다.

Spring Security가 적용되이 있습니다.

Kakao API를 이용한 소셜 로그인이 가능합니다.

개발 기간 : 2024/05/01 ~ 05/20

  • 캠핑장 예약
  • 마이페이지
  • 사업자 페이지

CakeDrama

ReactJS + Node.js

온라인 쇼핑몰 사이트 CakeDrama입니다.

ReactJS를 프론트엔드로 Node.js를 백엔드로 개발한 사이트입니다. Typescript를 적용하여 유지보수가 용이하도록 하였습니다.

react-router-dom의 useParams, useNavigate, Route, Router, Link, Routes를 사용했습니다.
